# Demo Script

1. Open `/` and show the live camera feed with the AR board overlay.
2. Click "Place board" and drag the four corners to align the board.
3. Make a move with click-to-move; highlight the move list update.
4. Switch to Practice Mode and make a move to trigger engine reply and coach feedback.
5. Open the Multiplayer panel:
   - Create game and copy ID
   - Join from a second tab and demonstrate move sync
6. Navigate to `/demos` and show the Hand Hygiene Trainer.
